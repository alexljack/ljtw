When youâ€™re writing code, one of the most common objects you'll come across is an array. Arrays are a great way to store a collection of data and come with many handy methods built in.

Arrays are zero-indexed, meaning that the first index (item in the array) is 0.
For example in the `animals` array below, `dog` is at index 0 and `cat` is at index 1.
Whilst the indexes of 'dog', 'cat', 'bird', 'tiger' are 0, 1, 2 and 3, the array still has a length of 4. Which we could confirm by writing `animals.length`.

```js
let animals = ['dog', 'cat', 'bird', 'tiger']

console.log(animals[0]) // 'dog'
console.log(animals[1]) // 'cat'
console.log(animals.length) // result will be 4
```

Often arrays will be used to store data in objects, so that they can be used store more information. Like below

```js
let animals = [
```
